# Trabalho-1-de-CG
* Geral: Desenvolver um Jogo Arcade ou uma Simulação (robótica, animal ou social) 2D. * Detalhes: - O programa deverá ter uma tela de abertura desenhada com os algoritmos de rasterização de reta, circunferência e elipse (os três devem ser usados). As figuras criadas devem ser preenchidas com o algoritmo Flood Fill ou com o Boundary Fill;
